# 2020-ithelp-contest

## 被選召的 Gopher 們，從零開始探索 Golang, Istio, K8s 數碼微服務世界

從單體服務一路到現在的微服務，Backend 經歷了不可思議的變遷，而 Golang 這短小精悍的語言就好像剛進到數碼寶貝世界的孩子，雖然一個人做的事情不多，但是只要相信 Kubernetes、Istio 等等數碼獸，就可以改變世界，本系列會以初學的角度出發，希望我與大家能有機會一起獲得微服務數碼徽章。
